# Lunarix-Bin
Created by  Team Lunara
Overall process : https://youtu.be/DrZt3cNTjXQ?si=G5j3BsIONoMT5cT_

## Overview
Lunarix Bin is a smart waste segregation system that automatically separates metal, non-metal and wet materials using sensors.

## Why I Built This
- Have you ever wondered just how many recyclable materials ended up in the trash bin instead of being recycled as they should?
- That's why my teammates and I built a smart bin that can automatically identify these materials!
  
## How It Works
- Detects material using sensors
- Sorts waste using a servo motor that will rotate the base

## My Role
- Leader of Lunara
- Designed the system
- Tested with real waste items
- Presented the prototype and wrote a research paper for this project

## Result
Successfully classified over more than 10 different-sized tins, papers, and soaked, i.e., ruined papers accurately.

## Reflection
This project taught me how basic engineering can be used to solve real-world environmental problems.
